자바스크립트 문장들은 웹 브라우저에게 내리는 명령이다.
주석은 // 사용.
변수 선언은 var, let, const (let은 최근에 사용, const는 변하지 않는 변수)

typeof를 사용해서 선언된 변수들의 자료형을 확인할 수 있음.
수치형(number, 선언할 때 실수 정수 상관없이 let 사용가능), 문자형(string), 부울형(boolean), 객체형(object), undefined(변수는 선언 되었지만 아직 값이 정해지지 않은 것)

템플레이트 리터럴
: 문자열 안에 변수를 호출할 수 있는 방법 ${x}와 같이 사용

연산자는 C언어의 연산자와 동일하게 사용
+ 연산자 +,-,*,/,% 계산하는 수행평가 볼 예정

비교연산자도 동일하게 사용
== : 값이 같으면 참
!= : 값이 다르면 참
> : 크면 참
< : 작으면 참
>= : 크거나 같으면 참
<= : 착거나 같으면 참

===연산자와 !== 연산자
=== : 값과 타입이 모두 같으면 참
!== : 값이나 타입이 다르면 참

논리 연산자도 C언어와 동일하게 사용
&&, ||

prompt() 함수
사용자에게 어떤 사항을 알려주고 사용자가 답변을 입력할 수 있도록 하는 함수 (기본적으로 문자열로 입력받음, prompt함수 앞에 parseInt 입력해주면 정수로도 입력받을 수 있음)

confirm() 함수
사용자에게 확인이나 취소 요구하는 창
